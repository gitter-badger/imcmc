===================
imcmc
===================
:imcmc: A C++ implemented MCMC library
:Version: 0.0.3
:Author: Youhua Xu
:Homepage: http://lss.bao.ac.cn/~xyh/

Description
============

This code aims to provide easy to use interfece to MCMC simulations.  Currently only the Affine-Invariant Ensemble Sampler algorithm has been implemented, in futher I will add other commonly used algorithms / samplers into this lib.
